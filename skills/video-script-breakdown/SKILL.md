# 视频脚本拆解与拉片分析

## 概览

使用本 skill 将脚本、视频文件、字幕稿、转写文本或参考视频描述，转化为结构化的制作分析。优先输出能直接用于拍摄、剪辑、AI 视频生成和复刻创作的内容：分镜表、时间轴节奏、人物环境动作、视听语言、复刻建议和提示词。

## 工作流程

1. 判断输入类型：
   - **只有脚本**：按场景、信息点、情绪转折、台词句群或故事节拍拆分。
   - **视频文件**：先查看时长、分辨率、帧率、音频轨等元信息；如工具可用，再抽取关键帧或片段，按时间码分析。
   - **字幕稿/转写文本**：把口播、旁白或对白对齐到画面节奏，并标出需要补充推断的视觉与声音信息。
   - **参考视频描述**：在信息不足但不影响主判断时直接分析；只有缺失信息会显著影响结论时再询问用户。

2. 选择输出深度：
   - **快速拆解**：提炼 5-10 个关键节拍，适合创意判断、选题分析或方向讨论。
   - **制作级拆解**：按镜头输出表格，包含时间码、画面、动作、景别、运镜、字幕、音频和执行建议。
   - **复刻包**：在拉片表基础上补充可复用提示词、分镜建议、素材清单和剪辑配方。

3. 当用户要求详细分析人物、环境、动作、运镜、景别、转场、字幕、声音、BGM、色调、节奏或爆款复刻时，读取 `references/analysis-dimensions.md`。

4. 如果用户需要空白模板或标准 Markdown 拉片表，运行：

```bash
python scripts/make_breakdown_template.py --shots 12 --mode remake --output breakdown.md
```

5. 分析真实视频文件时，优先进行确定性的媒体检查：
   - 使用 `ffprobe` 或媒体信息工具获取时长、帧率、分辨率、编码和音频轨。
   - 使用 `video-frames` skill 或 `ffmpeg` 抽取场景切换帧、关键帧或固定间隔帧。
   - 如果无法自动检测镜头切换，短视频按 2-5 秒间隔采样，长视频按 10-30 秒间隔采样。
   - 生成的帧图、片段和分析文件要命名清楚，并放在源视频附近或独立任务目录中。

## 输出格式

默认使用 Markdown。除非用户明确要求英文，否则全部用中文输出。

### 脚本拆解

包含：

- 核心主题 / 情绪曲线
- 受众与平台判断
- 结构拆解：开头钩子、铺垫、冲突/信息增量、高潮、收束/CTA
- 分镜表：镜头、画面、人物、环境、动作、台词/字幕、音频、时长
- 可复刻提示词：按镜头写，不只写整体风格
- 制作风险：难拍镜头、缺失素材、叙事不清、角色/场景连贯性问题

### 视频拉片

除非用户要求其他格式，否则使用以下表格列：

| 时间码 | 镜头 | 人物 | 环境/道具 | 动作 | 景别/构图 | 运镜 | 转场/剪辑 | 字幕/包装 | 音频/BGM | 色调/光线 | 节奏功能 | 复刻建议 |
|---|---|---|---|---|---|---|---|---|---|---|---|---|

表格后补充：

- **结构规律**：哪些元素在重复、递进、反转或节奏重置。
- **视听语言**：镜头、构图、色彩、字体、声音、剪辑如何共同制造效果。
- **爆点机制**：钩子、反差、好奇心、权威感、情绪、爽点或转化点。
- **复刻方案**：哪些可以照搬，哪些需要改写，哪些不建议照搬。
- **AI 生成提示词**：按镜头或场景给出图像/视频模型可用的提示词。

### 复刻包

面向 AI 视频或短视频复刻时，包含：

- 分镜级提示词，优先中文；如模型更适合英文，可额外给英文版。
- 负面提示词或规避说明，用于减少手指、面部、字幕、Logo、物体变形等问题。
- 素材清单：人物、场景、道具、字幕、音效、BGM、口播声音风格。
- 剪辑配方：每镜时长、转场方式、字幕节奏、音频进入/退出点。
- 渲染前质检清单。

## 分析规则

- 区分“观察到的内容”和“推测出的意图”；不确定处标注“推测”或“需确认”。
- 没有精确工具时，时间码可以估算，但要说明是约略时间。
- 不要只写“高级感”“电影感”等空泛词，要拆成可观察的画面选择：光线、构图、镜头、色彩、声音、节奏。
- 只有脚本时，不要强行虚构精确运镜；可以标为“推荐执行方式”。
- 商业内容、广告和爆款短视频，要把创作选择连接到留存、信任、情绪、转化或互动动机。
- AI 视频工作流中，要把抽象描述转化成具体视觉约束：主体、动作、场景、镜头、构图、光线、风格、时长、运动方式和连贯性锚点。
